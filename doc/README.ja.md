# 🧬 ComfyUI-CheckpointHandpickerSuite (日本語版聖典)

> ** 「ディスク容量が節約できる保証はありません。なぜなら、あなたが空いた容量に歓喜して、明日さらに多くのCheckpointをダウンロードしてしまうからです。」**

エクスプローラーとの果てしない往復も、不要モデルを消すためだけの不毛なComfyUIの再起動も、すべては過去の遺物じゃ。
`ComfyUI-CheckpointHandpickerSuite` は、膨大なCheckpoint（.safetensors）の海で戦うAI絵師のために、生成パイプラインを「一瞬も止めず」、常時稼働のままリアルタイムにモデルの格付け・棚卸しを執行するために錬成された、究極のノンストップ・アセット管理システムじゃ。

---

## ❔ これは何なのか？（What is this?）

**「あなたのCheckpoint読み込みノードに機能を追加し、豪華なジュークボックスにします。」**

大量のモデルをただの「肥大化したデータファイル」として無機質に扱う時代は終わりました。
このSuiteを導入した瞬間、あなたのComfyUI環境は、世界中の神モデルたちが織りなす「極上のAIアート・ジュークボックス」へと変貌します。

ユーザー（あなた）が行うのは、お気に入りの名盤（Checkpoint）を選ぶことだけ。
寝ている間に自動でレコードが切り替わり（Cycler）、
吐き出された極上のメロディ（画像）を目の前で楽しみながら（Preview）、
「お気に入り（💛）」のプレイリストを作るか、「棚卸し（🗑）」にするかを1クリックでジャッジする。

これは、ストレージ難民を救済し、モデルの命の選別をゲーム感覚で行うために生まれた、世界で最も贅沢な「アセット・ソムリエシステム」なのじゃ！

## 🗂 構成要素の説明（The Four Sisters）

我がリポジトリを支える、シャドウガーデンの序列（4人の看板娘たちと5つの魔導具）を紹介しよう。それぞれが固有の役割を持ち、あなたのComfyUI（肉体）と脳内で非同期にリンクしておるのじゃ！

### 1. 🤎 Checkpoint Name Cycler（心臓）

バックグラウンドで何百件ものジョブをノンストップで爆走させるための動力源じゃ。

* `fixed`（固定）、`increment`（順次）、`randomize`（ランダム）、そして全モデルを1枚のデッキにしてシャッフルし、重複なしで引き切る `shuffle_once` の4モードを完備。
* KSamplerへモデルを渡す直前のフェーズでCheckpoint名を動的に注入する「ジョブの実行時遅延解決（Lazy Resolution）」を搭載。
* さらに、現在のインデックスを汚さずに特定のモデルを最優先で割り込ませる緊急レーン「Local List」をも制御する、本Suiteの文字通りの「心臓」じゃ。

### 2. 💜 Checkpoint List Selector（頭脳）

リポジトリ内の全モデルを一手に統括し、状況に応じて役割を2変化させる司令塔じゃ。

* 接続先がない時は **`🏹 Push to Local List`** として機能し、Cyclerへ割り込み曲を送り込む。
* 右手（Tagger）や目（Preview）とリンクした瞬間、UI動相同期モードである **`🎯 Sync Checkpoint`** へと動的に覚醒する！
* ポチッと押すだけで、新モデルの追加や物理削除の結果を、**ComfyUIの再起動なしで1秒で同期する `🔄 Refresh All` アビリティ**も有しておるぞ。

### 3. 💛 Checkpoint Status Tagger（右手）

画像を見ながら、モデルに対して冷徹かつ瞬時に格付け（レイティング）を行う執行官じゃ。

* ボタンは `favorite(💛)` / `nice(👍)` / `keep(✔)` / `delete(🗑)` の4つ。（`none` はトグルで解除）。
* 特筆すべきは、「💛や👍がついている間は、deleteボタンが物理的にグレーアウトし、サーバー側でも絶対に削除予約を弾く」という、うっかり誤爆を100%許さない鉄壁のセーフティバリデーション。右手の誇りにかけて、神モデルを絶対に見捨てぬ設計じゃ！

### 4. 💙 Ephemeral Preview ＆ ImageDir Preview（目）

インジェクションされた情報とストレージをマッハで逆探知する、Suiteの「網膜」じゃ。

* **Ephemeral Preview:** 中間生成ラインで、今まさに画像を吐き出しているモデルの名前を、フロントエンドのタブローカルな実行状態から検知して美しくラベリング表示する。
* **ImageDir Preview:** 棚卸しモードの主役。Selectorでモデルを選んだ瞬間、過去にそのモデルが出力した画像（サブフォルダ含む）を高速スキャンし、浮動小数点誤差を完璧に排除した超最適化グリッド（コンタクトシート）としてキャンバスへ一瞬で展開してみせるのじゃ！

## 🗑 削除予約の最終執行について

Taggerで `delete（🗑）` を押したモデルは、その場では物理削除されず、安全のためにテンポラリの `checkpoint_delete_queue.jsonl` に予約キューとして蓄積される。
バッチ作業がすべて終わったら、自動生成されている安全執行スクリプト `delete_reserved_checkpoints.py` をコンソールから実行し、対話的に `(y/N)` でパージすればSSDの容量解放は完了じゃ！もちろん、ComfyUIの再起動は不要じゃぞ。
